# Глоссарий

Справочник терминов, используемых в проекте "Базис и Надстройка".

---

## Основные Понятия

| Термин | Определение |
|--------|-------------|
| **Базис** | Экономическая основа общества: производительные силы и производственные отношения. Определяет надстройку. |
| **Надстройка** | Совокупность идеологических, политических и культурных институтов, возникающих на основе базиса. |
| **Производительные силы** | Средства производства (орудия труда, технологии) и рабочая сила (люди с навыками). |
| **Производственные отношения** | Отношения между людьми в процессе производства: кто владеет, кто работает, как распределяется продукт. |
| **Формация** | Исторически определённый тип общества с характерным способом производства (первобытная, рабовладельческая, феодальная). |
| **Класс** | Социальная группа, определяемая отношением к средствам производства (владельцы vs работники). |
| **Классовое сознание** | Осознание NPC своего классового положения и интересов. Влияет на поведение и отношения. |

---

## Система NPC

| Термин | Определение |
|--------|-------------|
| **NPC** | Non-Player Character — персонаж, управляемый искусственным интеллектом. Каждый NPC уникален. |
| **Личность** | Набор черт характера NPC: экстраверсия, добросовестность, открытость, дружелюбность, невротизм. |
| **Потребности** | Внутренние состояния NPC (голод, усталость, одиночество), определяющие приоритеты действий. |
| **Характеристики** | Базовые параметры NPC: сила, ловкость, интеллект, харизма, здоровье. |
| **Навыки** | Приобретённые умения NPC: земледелие, охота, ремесло, торговля. Развиваются через практику. |
| **Отношения** | Связи между NPC: дружба, вражда, романтика, родство. Влияют на взаимодействия. |
| **Генетика** | Система наследования черт от родителей: характеристики и предрасположенности передаются детям. |

## BDI-Архитектура

| Термин | Определение |
|--------|-------------|
| **BDI** | Beliefs-Desires-Intentions — архитектура искусственного интеллекта для принятия решений. |
| **Beliefs (Убеждения)** | Что NPC знает или думает о мире: информация о ресурсах, других NPC, событиях. |
| **Desires (Желания)** | Цели и стремления NPC: выжить, разбогатеть, завести семью, стать мастером. |
| **Intentions (Намерения)** | Выбранные планы действий для достижения желаний на основе убеждений. |
| **Рефлексия** | Процесс осмысления опыта NPC: формирование выводов из воспоминаний. |

---

## Память и Поведение

| Термин | Определение |
|--------|-------------|
| **Поток памяти** | Хранилище всех воспоминаний NPC с метками времени и важности. Основа для рефлексии. |
| **Воспоминание** | Запись о событии в памяти NPC: что произошло, когда, с кем, насколько важно. |
| **Emergent поведение** | Сложное поведение, возникающее из простых правил. Не задаётся, а появляется само. |
| **Автономность** | Способность NPC самостоятельно принимать решения без внешнего управления. |
| **Приоритеты** | Ранжирование действий по важности на основе потребностей и личности NPC. |

---

## Мир и Симуляция

| Термин | Определение |
|--------|-------------|
| **Симуляция** | Процесс моделирования жизни общества во времени. Основной цикл игры. |
| **Тик** | Минимальная единица времени симуляции. Один час игрового времени. |
| **Локация** | Место на карте мира: поселение, лес, поле, шахта. NPC перемещаются между локациями. |
| **Ресурс** | Материальный объект в мире: еда, древесина, камень, руда, инструменты. |
| **Сезон** | Период года (весна, лето, осень, зима). Влияет на доступность ресурсов и активности. |
| **Катаклизм** | Случайное негативное событие: засуха, наводнение, мор, голод. Испытание для общества. |

## Экономика

| Термин | Определение |
|--------|-------------|
| **Собственность** | Право владения ресурсами и средствами производства. Может быть общинной или частной. |
| **Производство** | Процесс создания ресурсов и товаров из сырья с использованием труда и технологий. |
| **Технология** | Знание о способе производства или действии. Открывается через труд и передаётся обучением. |
| **Избыточный продукт** | Производство сверх необходимого для выживания. Основа для накопления и неравенства. |
| **Эксплуатация** | Присвоение результатов чужого труда владельцами средств производства. |

## Общество и Культура

| Термин | Определение |
|--------|-------------|
| **Семья** | Базовая социальная единица: родители, дети, родственники. Передаёт гены и знания. |
| **Демография** | Статистика населения: рождаемость, смертность, возрастная структура, миграция. |
| **Верование** | Представление о мире, не основанное на опыте: о духах, богах, сверхъестественном. |
| **Традиция** | Устойчивый паттерн поведения, передаваемый через поколения. |
| **Социальная норма** | Правило поведения, принятое в обществе. Нарушение влечёт санкции. |
| **Идеология** | Система идей, оправдывающая существующий социальный порядок. Часть надстройки. |

## Техническая Терминология

| Термин | Определение |
|--------|-------------|
| **Event System** | Система событий (pub/sub) для коммуникации между компонентами симуляции. |
| **Config** | Конфигурация симуляции: начальные параметры, скорости, вероятности. |
| **Generative Agents** | Подход Stanford к созданию правдоподобных агентов с памятью и рефлексией. |
| **CLI** | Command Line Interface — текстовый интерфейс управления симуляцией. |
| **GUI** | Graphical User Interface — графический интерфейс (pygame-ce). |

## Жизненный Цикл NPC

| Термин | Определение |
|--------|-------------|
| **Младенец** | Возраст 0-2 года. Полная зависимость от родителей. |
| **Ребёнок** | Возраст 3-12 лет. Обучение базовым навыкам. |
| **Подросток** | Возраст 13-17 лет. Формирование личности. |
| **Молодой** | Возраст 18-30 лет. Создание семьи, активный труд. |
| **Взрослый** | Возраст 31-50 лет. Пик производительности. |
| **Зрелый** | Возраст 51-65 лет. Накопленный опыт и мудрость. |
| **Старик** | Возраст 66+ лет. Повышенный риск смерти, передача знаний. |
